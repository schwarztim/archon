# Copilot CLI Instructions

## SDD Framework
This project uses copilot-sdd for spec-driven development.

### Workflow
1. Run `npx copilot-sdd check` to verify all goals
2. Read `.sdd/discovery/CODEBASE.md` before making changes
3. Read `.sdd/learnings/*.md` for known pitfalls
4. Follow specs in `.sdd/specs/` for feature work

### Agent Instructions
- Use the agent definitions in `.sdd/agents/` as system prompts
- Each agent has a specific role in the SDD pipeline
- The discover → spec → plan → execute → verify → learn flow ensures quality

_Learnings from previous sessions will be injected here automatically._
